# SOC Investigation Writeups

This repository contains sample SOC investigation scenarios including:

- Phishing investigations
- Malware analysis
- Failed login investigations
- Privilege escalation alerts
- Impossible travel detections

The purpose of this repository is to demonstrate:
- Incident response workflow
- Threat investigation methodology
- Detection analysis
- SIEM and log analysis skills
- MITRE ATT&CK mapping

> All logs, IP addresses, domains, and indicators used in these reports are for educational purposes only.
